## Install

```shell
git clone https://github.com/nelsonmestevao/dotfiles ~/.dotfiles
git clone https://github.com/nelsonmestevao/spells   ~/.spells
cd ~/.dotfiles
sh install.sh
```

## Uninstall

```shell
cd ~/.dotfiles
sh uninstall.sh
cd ~
rm -rf ~/.dotfiles
```
