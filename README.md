> ~ is where the :heart: is

## Install

```shell
git clone https://github.com/nelsonmestevao/dotfiles ~/.dotfiles
cd ~/.dotfiles
sh install.sh
```

## Uninstall

```shell
cd ~/.dotfiles
sh uninstall.sh
cd ~
rm -rf ~/.dotfiles
```
